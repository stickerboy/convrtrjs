## New features
### Search
- Quick search is now the default on all pages. The header link for search will open a sidebar with a search box, the box here works the same as the search page where typing will automatically generate a list of results. The advanced search link will take you to the results page including the query string of the text entered.

## Bug fixes
- Fixed color of filter icon in dark mode, which caused a duplicate icon to be visible
